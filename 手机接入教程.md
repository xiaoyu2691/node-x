### 3、下载成功后，需要安装些工具，请执行下方指令
pkg upgrade -y && pkg install proot -y && termux-chroot

### 4、执行指令接入hub

#### 接入指令：

直接运行：
curl -fsSL https://files.node-x.xyz/stor-setup-dev-test/install.sh | bash -s -- -token=token值 --foreground

后台运行：
curl -fsSL https://files.node-x.xyz/stor-setup-dev-test/install.sh | bash -s -- -token=token值
